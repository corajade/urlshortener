How to use this application:

  1. Add /new/www.yoururlofchoice.com to the end of this application's base URL
  2. Add the shortURL given to the end of the base URL to go to your chosen URL .

Examples:
Input=www.myherokuappURL.com/new/www.google.com
Output={"URL":"https://www.google.com, "shortURL":"3"}
Inputting: www.myherokuappURL.com/3 will redirect to www.google.com.
Input=www.myherokuappURL.com/new/cats.com
output="Invalid URL"
Input=www.myherokuappURL.com/xyz
Output=Redirect to another URL if this is a valid short URL or an error message if not valid.

How to use this application:

  1. Add /new/<yourUrlOfChoice> to the end of this application's base URL
    • If the url is a valid website, you will receive a short URL that will redirect to your chosen website
    • If the url is not a valid website, you will receive an error message indicating "invalid URL"
  2. To be redirected to your chosen website input the URL address given under the "shortUrl" into your browser's address bar.
    • All shortened URLs take the form https://powerful-chamber-28043.herokuapp.com/<a number and/or letter combination>
    • If you enter a number/letter combination that is not a shortened URL, you will receive an error message indicating invalid input

Examples:

  • Input=https://powerful-chamber-28043.herokuapp.com/new/www.google.com
    Output={"URL":"https://www.google.com, "shortURL":"https://powerful-chamber-28043.herokuapp.com/1"}
  • Input= https://powerful-chamber-28043.herokuapp.com/3
    Output= redirect to www.google.com.
  • Input=https://powerful-chamber-28043.herokuapp.com/new/Ilovecats
    Output="Invalid URL"
  • Input=https://powerful-chamber-28043.herokuapp.com/xyz
    Output=Redirect to another URL if this is a valid short URL or an error message if not valid.
